硬件#

以下内容为FinClip部署的推荐配置清单，通常我们建议主要组件尽可能分散部署，在资源有限且QPS规划考虑内适当合并组件所在节点。

需要注意的是，在很多情况下可以使用客户提供的自有服务，例如：

  • 若已有负载均衡器，可省略gateway
  • 条件允许情况下，存储服务推荐使用SSD磁盘
  • 若已有容器镜像仓库，可省略Harbor/registry
  • 若已有S3存储服务，可省略Minio
  • 若客户已有Kubernetes，可省略k8s Master x 3
  • 可根据实际用户数扩缩容
